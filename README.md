# Vesper theme for Fresh editor
### Vesper - a modern and clean color theme for [Fresh](https://github.com/sinelaw/fresh) (terminal text editor)

### Credits
Based on the great peppermint and orange flavored dark theme for Visual Studio Code (https://github.com/raunofreiberg/vesper)

### Install
Copy the vesper.json into .config/fresh/themes directory.

Then set the theme to vesper in .config/fresh/config.json.

```
{
  "theme": "vesper"
}
```
